The Zomic language is quite different from the vZome file format. Originally, vZome was nothing but a script window and a rendering window, and there was no editing supported at all. Zomic looks like this:

scale -1 symmetry through center symmetry { from repeat 4 size 6 yellow +0 from blue -13 branch { blue -13 blue +8 blue +7 orange +38 blue +7 orange +26 } blue +7 blue +13 repeat 2 { blue +7 blue +8 branch blue -13 } yellow +7 yellow +8 blue -13 }

Scott (scott@vorthmann.org)
2021-01-23 08:54:19

As a language, I’m rather proud of the design, but it is really hard to use nonetheless. See all those numbers? You have to have a key to know how to do the simplest thing.

I cannot recommend that you learn it, quite honestly.
